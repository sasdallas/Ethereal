Hardware Support

Ethereal was designed with large hardware support in mind. xHCI, NVMe, IDE, and more are all supported.

Full GUI

Ethereal contains a full graphical user interface - the window server and graphics library are almost fully custom. Bundled includes many graphical applications, like the terminal, font viewer, taskbar, etc.

Open Source

Ethereal is fully open source and licensed under the BSD 3-clause license.
